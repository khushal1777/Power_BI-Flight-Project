# Power_BI-Flight-Project
This dashboard analyzes key flight metrics such as delays, cancellations, routes, and airline performance. It provides clear, interactive visuals to help understand flight trends and improve operational decision-making.

Import the Flight Dataset

Use Get Data → Excel/CSV/SQL → load tables (Flights, Airports, Airlines).

Clean & Prepare the Data

Fix data types (Date, Time, Numbers).

Remove duplicates.

Create calculated columns if needed (e.g., Delay Minutes, Total Flight Time).

Create Relationships

Link tables:

Flights ↔ Airlines

Flights ↔ Airports (Origin & Destination)

Build DAX Measures

Total Flights

Total Delay Minutes

Average Delay

On-Time Percentage

Cancellation Count

Design Visuals

Map: Flights by origin/destination airports

Bar Chart: Delays by airline

Line Chart: Monthly/Yearly flight trends

Pie/Donut: Cancellation reasons

Cards: Key KPIs (Total Flights, Avg Delay, Cancellations)
